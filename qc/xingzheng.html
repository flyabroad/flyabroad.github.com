魁北克省行政区划

魁北克全省被分为17个行政地区，分别是：

• 下圣劳伦斯（Bas-Saint-Laurent）
o 位于魁北克省东南部，圣劳伦斯河南岸，面积22185平方公里，人口20.1882万（2005），下辖8个县（MRC），133个城镇，2个印第安保留地。

• 萨格内-圣约翰湖（Saguenay–Lac-Saint-Jean）
o 位于魁北克省中南部，圣劳伦斯河北岸，以着名的圣尚湖与山更尼湾等自然风光闻名，是魁北克省着名的集自然与人文于一体的旅游胜地。面积95893 平方公里，是魁北克省第三大行政区。人口27.5427万，下辖4个县（MRC），一个独立市山更尼，总共约60个城镇，1个印第安保留地。

• 首都区（Capitale-Nationale）
o 位于魁北克省中南部，圣劳伦斯河北岸，魁北克省省府魁北克市所在地。魁北克省9个历史纪念地区的5个位于该地区。面积18639平方公里，人口66.307万(2004)。下辖6个县（MRC），总计67个城镇，一个印第安保留地，一个教区，一个独立市魁北克市。

• 摩丽丝区（Mauricie）
o 位于魁北克省中南部，圣摩丽丝河与圣劳伦斯河交汇处，即摩丽丝国家公园所在地，魁北克省着名旅游胜地。面积35452平方公里，人口25.9424万（2004）。下辖3个县（MRC），三个地区，2个独立市，三个印第安保留地，总计拥有47个城镇。三河市位于该地区。

• 艾第区（Estrie）
o 位于魁北克省最南部，与美国接壤，是魁北克省着名的旅游区东方区 ( Les Cantons de l’Est ) 所在地。面积10195平方公里，人口30.0917万（2005），下辖6个县（MRC），一个独立市舍尔布克( Sherbooke )。

• 蒙特利尔（Montréal）
o 蒙特利尔（Montréal）也叫满地可，位于魁北克省西南部的圣劳伦斯河中的奥雪来嘉 ( Hochelaga ) 列岛的数个，是魁北克省面积第二小的地区，也是魁北克省最大城市蒙特利尔所在。面积498平方公里，人口158.0494万（2006）。

• 乌塔韦区（Outaouais）
o 位于魁北克省西部，乌塔韦河北岸，与加拿大首都渥太华以及安大略省为邻。面积30504平方公里，人口33.8491万，下辖4个县（MRC），1个独立市嘉丁诺 ( Gatineau )，总计75个城镇。

• 阿比蒂比-蒂米斯坎明格（Abitibi-Témiscamingue）
o 位于魁北克省最西部，紧邻安大略省，以林业，采矿业闻名。面积57340平方公里，人口14.4835万（2006）,下辖4个县（MRC），6个印第安保留地，总计79个城镇。

• 北岸（Côte-Nord）
o 位于魁北克省最东部，包括了圣劳伦斯河海湾北岸大部分地区。西邻山更尼地区，东邻纽芬兰与拉布拉多省，是魁北克面积第二大的地区，经济上主要依靠 铁矿，铝业以及林业。面积23.67万平方公里，人口96861(2004),下辖5个县（MRC），5个独立市，9个印第安保留地。山更尼河口处的泰道沙克是世界着名的观赏野生鲸鱼的胜地。

• 北魁北克（Nord-du-Québec）
o 位于魁北克省北部，包括了拉布拉多半岛的大半以及魁北克省陆地面积的55%。西部是James以及哈德逊海湾，北部是哈德逊海峡与安加法 ( Ungava ) 湾，东北为拉布拉多。该地区总面积83.9万平方公里，占魁北克省总面积55%。其中12.1万平方公里为河流和湖泊。人口39892人 (2004)。下辖2个半自治区和5个城镇。魁北克省水电公司在该地区拥有众多水电工程。

• 嘉比茜-马德兰岛（Gaspésie-Îles-de-la-Madeleine）
o 位于魁北克省东部，圣劳伦斯河海湾南岸的最东端，包括嘉比茜 ( Gaspesienne ) 半岛和 马德兰 ( Madeleine )列岛两个组成部分。经济主要是农业（盛产苹果，橄榄，葡萄等），渔业，林业以及旅游（着名的穿孔石贺雪卑西 ( Rocher perce ) 鸟岛）。该地区80%复盖着针叶林，也是魁北克省森林复盖率最高的地区。面积20272平方公里，人口96924人(2001)，下辖5个县（MRC），2个独立市，2个印第安保留地，总计53个城镇。

• 肖迪耶-阿帕拉什（Chaudière-Appalaches）
o 以境内的肖迪耶 ( Chaudiere )河与阿帕拉什 ( Appalaches )山脉命名，位于魁北克省南部，圣劳伦斯河南岸，与魁北克市隔河相望。经济上主要是林业，造纸，农业，石化，冶金，旅游以及纺织业。该地区拥有6000多 家农场，是魁北克省第二大农业区。矿产包括石棉，砂石等。旅游则以多种多样的服务闻名。面积15071平方公里，人口39.3415万（2004），下辖 9个县（MRC）,一个独立市利费 ( Levis ) ， 总计136个城镇。

• 拉华尔（Laval）
o 位于大蒙特利尔地区北部，隔拜日耶 ( Prairies ) 河与蒙特利尔岛相望。Laval地区包括Jesus岛和Laval岛。该地区人口36.8709万（2006），面积246平方公里。洛华尔区拥有众多高 科技工业园区，许多制药企业云集该地区。现在洛华尔区也是大蒙特利尔地区重要的商业中心。

• 勒奴地耶（Lanaudière）
o 位于蒙特利尔东北，面积13537平方公里，人口42.4233万。下辖6个县（MRC），一个印第安保留地。地势上南地北高，从最南部的圣劳伦 斯河谷到北部的洛康恩山地海拔高度逐渐升高。勒奴地耶区的旅游资源以文化和自然环境的和谐而着称，是魁北克省着名的度假胜地。勒奴地耶区的得名来自为纪念 领主洛发提的女儿玛日‧沙洛‧迪‧勒奴耶( Marie-Charlotte de Lanaudiere )。

• 洛朗第（Laurentides）
o 以境内的洛康恩山脉得名，拥有众多湖边别墅和滑雪场，是着名的休闲旅游胜地。该地区紧靠蒙特利尔，位于圣洛唐斯河北岸。面积20560平方公里，人口49.9882万，下辖8个县（MRC），一个印第安保留地，总计83个城镇。

• 蒙泰雷吉（Montérégie）
o 以境内的蒙泰雷吉山系得名，蒙特利尔皇家山是该山系最着名的一座。位于魁北克省西南，圣劳伦斯河南岸，南与美国接壤。大蒙特利尔地区的朗格惠属于 该地区。面积11111平方公里，人口137.1731万，下辖14个县（MRC），5个独立市，2个印第安保留地，总计176个城镇。经济上主要依靠农 业和旅游业。

• 中央魁北克市（Centre-du-Québec）
o 1997年7月30日正式设立。位于魁北克省中南部莫西西地区南部，与三河市隔圣劳伦斯河相对。该地区以魁北克面包篮和波瓦弗朗（Bois- Francs）闻名，出产各种谷物，蔬菜，水果，奶制品以及硬木木材。面积7332平方公里，人口22.6856万（2005），下辖5个县（MRC）， 2个自治区，总计89个城镇。
