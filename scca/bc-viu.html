温哥华岛大学（Vancouver Island University）学校概况：

温哥华岛大学（Vancouver Island University）是一所公立大学，创建于 1969  年，原名马拉斯比纳大学(Malaspina University College，或叫玛拉斯皮纳大学)，2008 年更名为温哥华岛大学。温哥华岛大学位于世界著名的风景区 BC省 的温哥华岛。该校可以授予学士学位，并与邻近的名牌院校英属哥伦比亚大学、维多利亚大学和西蒙弗雷泽大学结有伙伴关系，多个专业三、四年级学生可转入这些名校。温哥华岛大学 2008 年学生数超过 19,000，包括数百名各国留学生。学校所处的那奈谟市（Nanaimo）华裔人口只有数百人，因此这里对中国留学生学习英语，具有得天独厚的环境。

温哥华岛大学（Vancouver Island University）所授专业：

  1. 大学学位课程 文科学士（商业﹑写作﹑政策研究等） 少儿和青年护理学士 教育学士 理学士（生物） 理学士（护理） 旅游管理学士
  2. 与其它大学合办的课程 商业学士 ( Royal Roads 大学 ) 计算机学士 ( Victoria 大学 )
  3. 大学转学课程（大学前两年）文科学士 商业管理学士 饭店管理学士 体育教育学士 剧院和视觉艺术学士 古典音乐学士 理科学士 应用科学学士（工程）
  4. 餐饮课程 烘焙证书 厨师证书 饭店管理文凭
  5. 商业课程 商业文科学士 商业管理文凭（会计﹑金融、管理﹑营销） 计算机文凭办公室管理证书 工艺和表演艺术课程 图表﹑室内设计﹑视觉艺术文凭 爵士音乐文凭 剧院文凭
  6. 人类服务课程 少儿和青年关心文凭 娱乐管理文凭 早期幼儿教育和护理文凭 旅游研究文凭

温哥华岛大学（Vancouver Island University）热门专业点评：

  1. 热门专业1：工商管理硕士（MBA）；学费：21000加币；学历文凭：硕士；学制：12 个月；入学年龄：18；语言要求：托福550分。该校的MBA专业是与英国历史悠久的哈佛德舍尔大学合作推出的项目，为您在第一时间投身商界作最快捷的准备。
  2. 热门专业2：商学（Commerce）；学费：11000加币；学历文凭：本科；学制：4；    入学年龄：18；语言要求：托福580分。四年制学位课程，与多个名校合作。学生在马拉斯比纳大学（现在的温哥华岛大学）完成1，2年级课程之后，根据平均分和本人意愿，3，4年级可以注入英属哥伦比亚大学，维多利亚大学，西蒙菲莎大学，皇家路大学在那里完成学业，病获得毕业学院的学士学位。
  3. 热门专业3：旅游管理（Tourism Management）；学费：11000加币；学历文凭：本科；学制：4；    入学年龄：18；语言要求：托福580分。四年制学位课程，含带薪实习。它为学生提供旅游娱乐业在服务管理方的应用技能和概念。

温哥华岛大学（Vancouver Island University）综合入学条件：

本科课程： 高中毕业，托福 550 分或 IELTS6.5 以上，可直接入读本科课程。无英文成绩，可获有条件录取，完成马拉斯比纳大学学院的大学预科英语证书课程后，入读本科。有些专业需提交作品。最终录取结果以校方评估结果为准。
