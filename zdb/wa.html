澳大利亚西澳（WA）政府担保对申请者（190州担保、489地区担保）的要求 - FLYabroad

西澳州担保申请要求：职业需要在 Western Australia’s State Priority Occupation List (SPOL) 上的申请人就有机会申请移民，如果职业在本 WASMOL（Western Australian skilled migration occupation list） 上，那么申请190 (Skilled nominated) 没有任何额外要求，只需要通过职业评估，雅思4个6，不算州担保的5分达到55分就也可以申请。

如果职业在本 WASMOL 且申请 489 (Skilled Regional)签证，则需要当前仍在从事提名职业相关的工作，或者在西澳偏远地区有相关职业的job offer或者在该区域找到3个相关职业的工作机会（找3个招聘信息）。

对于在西澳的留学生需要持有有效签证且找到工作offer，这个offer还需要持续12个月才有效，也就是刚拿到一个12个月的offer，或已经在西澳从事提名相关职业，但该工作到期还有12个月以上时间。

对于不是西澳的留学生，或者已经在澳洲其他州有居住权的申请人（例如申请了SA的489想申请WA的190）也必须在西澳找到12个月以上提名职业相关工作offer才可以。

在Western Australia’s State Priority Occupation List (SPOL) 上但不在本 WASMOL 上的职业叫做 OFFLIST 职业，OFFLIST职业申请190需要有12个月的雇主offer才可以。OFFLIST 职业申请489需要在region有12个月提名职业相关的工作offer且证明对西澳已经有详细的了解（这个就比较软性了）。

可见留学生移民并不那么容易，需要在该州找到提名职业相关的全职且长度超过12个月的offer；可见，如果已经在澳洲境内，申请其他州的担保大概需要有雇主offer才可以，否则州政府会怀疑你去该州的诚意和必要性；这从侧面也说明，如果你移民成功后不再该州定居州政府也没有什么办法强制你在那。

2012年9月21号，西澳更新了担保职业清单，将之前已经关闭的职业去掉之外，同时增加了一些新的职业，其中包括一些常见的职业，比如交通工程师，ICT工程经理，生化工程师，材料工程师，项目管理员,房地产代表等，2013年10月28日删除了28个职业，当然同时添加了 72 个职业，详见：http://bbs.flyabroadvisa.com/index.php/topic,6707.0.html 。
