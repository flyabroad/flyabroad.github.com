东康乃狄格大学（Eastern Connecticut State University）- 威利曼蒂克(Willimantic) - 康乃狄格（Connecticut）- 美国

东康乃狄格大学(Eastern Connecticut State University)学校简介

东康乃狄格大学(Eastern Connecticut State University)位于历史著名的新英格兰地区，是一间严格筛选入学学生的教育机构，提供杰出的大学文科与理科课程，以及图书的专业与硕士课程。东康乃狄格大学每年招收 3,700 位全日制的大学学生，而小班制教学以及注重个人的学习气氛则是该校另一个傲人之处。学校致力于创造全球化社区，在国际学术界享有极高的校誉。Eastern的学生来自于世界30个以上不同的国家。Eastern良好的教育与社交风气为这些学生提供了个性发展的绝佳平台。弘扬文化多样性以及将学生视为独立个体并关怀备至，是学校百年以来不变的传统。Eastern的每位教职员都能够为学生提供个别的关心，以期满足国际学生的特殊需要。

东康乃狄格大学校校区位于康乃狄格州的Willimantic，距离该州的首府哈Hartford 只有 30 分钟的路程。到马萨诸塞州的波士頓只需要1.5 小时的车程，到纽约市只要2 小时的车程，不需一个小时即可抵达罗德岛和康乃狄格的沙滩

东康乃狄格大学(Eastern Connecticut State University)专业设置

会计，美国研究，生化，生物学，企业管理，信息系统管理，传播，电脑科学，教育，经济，英文，环境地球科学，历史，人文科学，数学，表演艺术，体育，政治科学，心里学，社会科学，社会工作，社会学，西班牙文学，运动与休闲管理，艺术创作，视觉艺术等

东康乃狄格大学(Eastern Connecticut State University)留学所需费用（学费+生活费）/年

  1. 本科：22，000美元
  2. 研究生：21，000美元
  3. 可申请奖学金（5，000-10，000美元）   

东康乃狄格大学(Eastern Connecticut State University)入学条件

  1. 本科：TOEFL 550
  2. 研究生：TOEFL550，GRE部分专业要求

东康乃狄格大学(Eastern Connecticut State University)学校网址：www.easternct.edu 。
