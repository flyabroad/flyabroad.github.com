爱达荷州立大学(Idaho State University)学校简介

爱达荷州立大学创立于1901年，是爱达荷州第2大的州立大学，优质的教育在2006年获得『美国新闻与世界报导』排名为全国第四级的学校，也是全美顶尖的公立学校之一。全校学生人数约13,000人，国际学生约350人来自65个国家，华人比率低。师生比率为1：17。校内有184个学生社团，也有中国同学会及亚洲同学会。学校整年有各种文化及体育活动；戏剧、艺术、音乐节目及各种研讨会。校内的电影院经常放映免费的电影供学生观赏。国际学生办公室提供免费接机、新生训练、文化及节庆活动、还有专人辅导课程及学习方法，对国际学生提供完善的服务。住宿可安排在校内的宿舍或公寓，也可以选择当地的接待家庭。

爱达荷州立大学(Idaho State University)位于波卡特洛（Pocatello）市，是一座人口约6万人的大学城，在爱达荷州的东南方，以民风纯朴、生活机能便利、安全和善的居民及低廉的生活费，名列在『富斯比杂志』选出的“美国10个最好的小城市”之内。

爱达荷州立大学(Idaho State University)专业设置

爱达荷州立大学(Idaho State University)设有副学士、学士、53个硕士学位及13个博士学位，设有文学院、理学院、商业院、工学院、教育学院、健康专业学院、科技学院、及药学院。包括了艺术，管理，教育，健康管理，医学，核能工程，电子工程，计算机科学，软件工程，物理学，数学，经济学等专业，其中尤以会计学最为著名。

爱达荷州立大学(Idaho State University)留学所需费用（学费+生活费）/年

  1. 本科：20，000美元
  2. 研究生：21，000美元
  3. 可申请奖学金（5，000-10，000美元）

爱达荷州立大学(Idaho State University)留学生入学条件

  1. 本科：TOEFL 500
  2. 研究生：TOEFL550 GRE/GMAT部分专业有要求
  3. 提供ESL语言课程（可免语言成绩申请）
