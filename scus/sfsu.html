旧金山州立大学（San Francisco State University）- 旧金山 - 加州 - 美国

旧金山州立大学（San Francisco State University）学校简介

旧金山州立大学位于加利福尼亚州（California，加州）旧金山市（San Francisco）区西南角的106公顷校地，为一所综合性大学，成立于1899年，是美国西岸最大的教育联盟－加州州立大学系统，23所分校的一所，由于位于旧金山得天独厚的地理位置，美丽的都会型大学校园位于非常安静、安全的住宅区，拥有便利的大众交通系统，前往市中心区也仅需15分钟左右的车程。校园旁边就是一个大型的购物商场，有许多精品店、餐厅、电影院等，一英哩之外就是太平洋岸。该校总共有约3万名学生，其中包括来自100多国的国际学生约2,200人。在加州州立大学系统中排名数一数二。除了有90亩的校地，还有丰富的硬设备及学校资源，校内有200多个学生团体、电视台、广播台和报社，为学生提供了培养多种兴趣的环境。

旧金山州立大学（San Francisco State University）专业设置

旧金山州立大学（San Francisco State University）有超过100个以上的大学专业及近100个研究生专业。包括工商管理，市场营销，社会学，英语教学，历史学，人力资源管理，音乐，体育教育，北美研究，生物化学，生物工程，广播与电子通讯，比较文学，娱乐业管理，工艺设计，心理学，数字媒体，表演艺术等。

旧金山州立大学（San Francisco State University）留学所需费用（学费+生活费）/年

  1. 本科：27，000美元
  2. 研究生：25，000美元
  3. 可申请奖学金（5，000-10，000美元）

旧金山州立大学（San Francisco State University）留学生入学条件

  1. 本科：TOEFL 500
  2. 研究生：TOEFL550
  3. 提供ESL语言课程（可免语言成绩申请）

旧金山州立大学（San Francisco State University）学校网址：www.sfsu.edu 。
