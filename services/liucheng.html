FLYabroad 签证工作室签证代理合作流程

基本流程：评估->签约->付费后开始材料处理并指导客户准备材料（有详细的材料清单）-> ->递档材料准备完成后FLYabroad可以代寄（也可以FLYabroad整理完成后由申请人自寄）。

中间FLYabroad会负责所有文字材料的制作，指导客户准备其他相关材料（成绩单，学时证明，存款证明，无犯罪证明，汇票办理等），解决客户疑难问题，客户只需要配合就可以了，十分省心。

如果以前申请过这种签证或者有拒签史，需要将以前递交的材料提供给FLYabroad。

加拿大联邦技术移民

填写评估表格进行评估（如之前申请过，需要附上之前的清单与申请表格） => 评估有90%以上的希望后签订协议 => 付费后，根据申请人的具体情况整理详细的客户需要进一步提供的材料清单 => 指导申请人搜集资料并帮助申请人准备申请必须的书面资料 => 申请人将资料快递（送至）签证工作室 => FLYabroad 整理材料并递交加拿大境内 CIO => 收到补料信后 FLYabroad 负责准备后期补料材料 => 递交具体使馆（北京、香港或其他使馆） => FLYabroad 指导客户准备背景调查，跟踪整个流程并处理中间可能发生的问题 => 体检 => 签证。

魁省技术移民

填写评估表格进行评估，有80% 以上希望的签订协议 => 付费后，根据申请人的具体情况整理材料清单，指导申请人搜集资料 => FLYabroad准备申请必须的书面资料，代做认证，根据情况代做公证 => 申请人将材料清单中要求的资料（例如存款证明，申请费汇票，无犯罪证明等）快递（送至）签证工作室，FLYabroad整理完善后递交使馆（可FLYabroad代寄也可以申请人自己寄） => 档案号后发魁省相关资料（中法版）给申请人 => 排期信后发面试相关资 => 面试信后发针对每个人的面试辅导资料，更新完善面试携带资料 => 面试=> 拿到 CSQ后准备联邦资料—FN—体检表 => 签证（整个申请过程签证工作室跟进）

澳洲技术移民

填写评估表格进行评估 => 有80%以上希望的签订协议 => 付费后，根据申请人的具体情况整理职业评估清单 => 指导申请人搜集资料 => FLYabroad负责准备评估必须的书面资料及具体资料 => 申请人将FLYabroad要求的资料快递（送至）签证工作室—完善递交职业评估的申请并跟进评估的进度 => 准备州担保申请（州担保类签证，如176，475） => 准备签证的相关资料，在线申请，并贴附相应的资料—跟进申请的进度，体检，预签，准签，贴签（全程与评估机构，州政府，使馆协调与沟通）

加拿大团聚移民

填写评估表格进行评估（如之前申请过，需要附上之前的清单与申请表格，拒签函，面试问题） => 真实婚姻的签订协议 => 付费后，根据申请人的具体情况整理资料（补充资料）清单，指导申请人搜集资料=> FLYabroad负责准备申请必须的书面资料 => 申请人将资料快递（送至）签证工作室 => FLYabroad汇总整理完善 => 申请人体检 => 申请资料递交密西沙加担保中心 => 担保资格通过后转回境内使馆—FN—面试信 => 提供周全的面试辅导，确保申请人面试通过 => 签证

探亲签证

填写评估表格进行评估（如之前申请过，需要附上之前的清单与申请表格） => 评估有50%以上的希望后签订协议 => 付费后，根据申请人的具体情况整理详细的客户需要进一步提供的材料清单 => 指导申请人搜集资料并帮助申请人准备申请必须的书面资料 => 申请人将资料快递（送至）签证工作室 => FLYabroad汇总整理完善2日内递交签证处理中心—3-5个工作日出结果。

加拿大商业移民

魁省投资：填写评估表格进行评估 => 有80%以上希望的签订协议 => 根据申请人的具体情况制定申请方案，列申请中涉及到的材料具体清单 => 指导申请人搜集资料 => 申请人将资料快递（送至）签证工作室—协助申请人准备书面资料以及其他商业经营资料 => 完善申请递交香港移民局—档案号 => 补料信/排期信 => 面试信，协助准备面试更新资料—投资款—CSQ—联邦申请—FN—ME—签证

联邦投资：填写评估表格进行评估 => 有80%以上希望的签订协议 => 根据申请人的具体情况制定申请方案—递交申请简表 => FN => 列申请中涉及到的材料具体清单 => 指导申请人搜集资料 => 申请人将资料快递（送至）签证工作室—协助申请人准备书面资料以及其他商业经营资料—补交使馆具体资料—面试/免面试 => 体检—投资款—签证
